Woocommerce price and products accordion panel view filtered by From-To price with ajax load more option.
 
== Description ==

Tabular Price Pane is a best plugin for the Home and Sub-Pages that is used to enhance the shopping experience and view the products more attractive with prices accordion panel for woocommerce in wordpress.

Customer can search and view the products according to 'From' and 'To' Price. This plugin supports unlimited shortcodes and widget for sidebar and content area and fully responsive with four different styles.


= Free Features =

* Load the products using Ajax as per 'From Price – To Price' panel selection.
* Fully responsive product grid and customizable template.
* Provides four modern and clean four different styles.
* Supports unlimited widget for sidebar and content area on the same or any page.
* Ajax based product view with 'load more' paging to load next products.
* Select color from colorpicker for background and text like, widget heading, product price, title, price tab.
* Allows to select default category from admin panel to load only products of that category and show/hide the search area if not needed for particular pages. 
* Set the dynamic 'From', 'To' and 'Difference' prices for accordion panel configuration.
 
= Premium Features =

* Search functionality - View the products searched by categories and product title. 
* Supports unlimited shortcodes for the various content area on the same or any page.

== Installation ==

1. Upload and extract the zip file downloaded to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Configure widgets from Appearance > Widgets and assign to page sidebar.
